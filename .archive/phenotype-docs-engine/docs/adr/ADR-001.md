# ADR-001: Documentation Engine Architecture

## Status
**Accepted**

## Context
Phenotype projects need consistent, high-quality documentation. A documentation engine must support multiple output formats, themes, and content sources while remaining performant.

## Decision
We will build a VitePress-based documentation engine with custom plugins:
- VitePress as static site generator
- Custom theme system for Phenotype branding
- Multi-source content aggregation
- Plugin architecture for extensions
- Built-in search with Algolia integration

## Consequences
- Fast builds with Vite
- Vue-based customization
- Consistent branding across docs
- Learning curve for Vue
