# ADR-002: Documentation Theme System

## Status
**Accepted**

## Context
Consistent branding and navigation across all Phenotype documentation requires a unified theme system.

## Decision
- Base theme on VitePress default
- CSS custom properties for theming
- Dark/light mode support
- Component library for common patterns
- Per-project theme customization options

## Consequences
- Visual consistency
- Accessible theming
- Maintenance overhead
