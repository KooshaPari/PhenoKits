# ADR-004: Build Pipeline Architecture

## Status
**Accepted**

## Context
Documentation builds must be fast, reliable, and support multiple output formats.

## Decision
- VitePress for HTML generation
- Playwright for PDF generation
- CI/CD integration
- Incremental builds
- Multi-format output (HTML, PDF, JSON)

## Consequences
- Fast incremental builds
- Multiple output formats
- Build infrastructure complexity
