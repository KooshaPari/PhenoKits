# ADR-005: Search Architecture

## Status
**Accepted**

## Context
Effective search is critical for documentation usability.

## Decision
- FlexSearch for small sites
- Algolia DocSearch for main portal
- Multi-language search support
- Search result ranking
- Offline search capability

## Consequences
- Fast search experience
- Algolia dependency for main portal
- Indexing overhead
