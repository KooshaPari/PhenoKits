# ADR-001: Gauge Metrics Collection Architecture

## Status
**Accepted**

## Context
Phenotype applications need comprehensive metrics collection for monitoring, alerting, and optimization.

## Decision
- Prometheus-compatible metrics endpoint
- Client libraries for Rust, Go, Python, TypeScript
- Automatic instrumentation for HTTP/gRPC
- Custom metric registration API
- Histogram and summary support
- Label-based dimensionality

## Consequences
- Industry standard compatibility
- Easy Grafana integration
- Cardinality management needed
