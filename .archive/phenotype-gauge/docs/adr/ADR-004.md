# ADR-004: Aggregation and Rollup Strategy

## Status
**Accepted**

## Context
Long-term metric storage requires aggregation to manage costs while preserving query capabilities.

## Decision
- Raw retention: 7 days
- 5-minute rollup: 30 days
- 1-hour rollup: 1 year
- Recording rules for common queries
- Downsampling with statistical accuracy
- Tiered storage architecture

## Consequences
- Reduced storage costs
- Queryable long-term trends
- Loss of raw granularity over time
