# ADR-005: Alert Integration

## Status
**Accepted**

## Context
Metrics must integrate with alerting systems for operational notifications.

## Decision
- Prometheus Alertmanager integration
- Rule definition in YAML
- Severity levels: critical, warning, info
- Alert templating
- Multi-channel routing
- Alert grouping and silencing

## Consequences
- Industry standard alerting
- Flexible routing
- Complex rule management
