# ADR-002: Request Context Propagation

## Status
**Accepted**

## Context
Request-scoped context must flow through middleware for tracing, logging, and auth.

## Decision
- contextvars for context storage
- Async context propagation
- Automatic child context inheritance
- Thread-local fallback for sync code
- Context serialization for logging
- Type-safe context keys

## Consequences
- Clean context access
- Works across async boundaries
- Some overhead
