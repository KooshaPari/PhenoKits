# ADR-003: Authentication Middleware

## Status
**Accepted**

## Context
Unified authentication across Python services is essential for security.

## Decision
- JWT token validation
- WorkOS AuthKit integration
- Token refresh handling
- Role-based access control
- Anonymous request support
- Auth context propagation

## Consequences
- Consistent auth across services
- Dependency on WorkOS
- Token management complexity
