# ADR-004: Logging Middleware

## Status
**Accepted**

## Context
Structured logging with request context improves observability.

## Decision
- structlog for structured logging
- Request ID injection
- Log level per endpoint
- Sensitive data redaction
- Performance metrics logging
- Correlation ID propagation

## Consequences
- Queryable logs
- Security through redaction
- Performance overhead
