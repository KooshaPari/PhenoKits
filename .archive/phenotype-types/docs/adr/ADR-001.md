# ADR-001: Type System Architecture

## Status
**Accepted**

## Context
Cross-language type sharing enables type-safe communication between services written in different languages.

## Decision
- Protocol Buffers as canonical type definition
- Code generation for Rust, Go, Python, TypeScript
- JSON Schema derivation for validation
- Versioned type packages
- Breaking change detection
- Type registry for discovery

## Consequences
- Type safety across services
- Code generation complexity
- Version synchronization challenges
