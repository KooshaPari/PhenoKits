# ADR-002: Code Generation Strategy

## Status
**Accepted**

## Context
Generated code must be idiomatic and maintainable.

## Decision
- Prost for Rust
- Protoc-gen-go for Go
- MyPy protobuf plugin for Python
- ts-proto for TypeScript
- Custom options for metadata
- Generated code in version control

## Consequences
- Idiomatic APIs per language
- Large generated files in repos
- Regeneration on proto changes
