# ADR-004: Validation Integration

## Status
**Accepted**

## Context
Types need validation constraints beyond protobuf basics.

## Decision
- Custom options for constraints
- Generated validation code
- JSON Schema validation
- Custom validators
- Validation error messages
- Validation skipping for trusted sources

## Consequences
- Rich validation
- Code generation complexity
- Runtime overhead

## References
1. [JSON Schema Validation](https://json-schema.org/)
