# ADR-005: Type Registry and Discovery

## Status
**Accepted**

## Context
Developers need to discover available types across the ecosystem.

## Decision
- Central type registry
- Searchable type catalog
- Type relationships graph
- Usage examples
- Breaking change notifications
- Type usage analytics

## Consequences
- Improved discoverability
- Registry maintenance overhead
- Dependency on central service
