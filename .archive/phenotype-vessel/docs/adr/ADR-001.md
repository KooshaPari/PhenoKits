# ADR-001: Vessel Container Orchestration

## Status
**Accepted**

## Context
Lightweight container orchestration for development and edge deployment.

## Decision
- Docker Compose compatible
- Kubernetes subset support
- Resource constraints enforcement
- Service mesh integration
- Hot reload for development
- Multi-node clustering

## Consequences
- Developer-friendly
- Production-capable
- Simpler than full K8s
