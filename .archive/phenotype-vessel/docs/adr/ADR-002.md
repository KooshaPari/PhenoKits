# ADR-002: Resource Management

## Status
**Accepted**

## Context
Efficient resource allocation and limits for containers.

## Decision
- CPU shares and quotas
- Memory limits with swap control
- Disk I/O throttling
- Network bandwidth shaping
- cgroup v2 support
- Resource monitoring

## Consequences
- Predictable performance
- Noisy neighbor prevention
- Complexity in tuning
