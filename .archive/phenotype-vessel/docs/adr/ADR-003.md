# ADR-003: Networking Model

## Status
**Accepted**

## Context
Container networking must be simple yet flexible.

## Decision
- Bridge networking default
- Service discovery via DNS
- Load balancing for replicated services
- Network policies for isolation
- Port forwarding for development
- Overlay for multi-node

## Consequences
- Works out of the box
- Production-ready
- Some platform limitations
