# ADR-004: Storage and Persistence

## Status
**Accepted**

## Context
Data persistence for stateful services.

## Decision
- Named volumes
- Bind mounts for development
- Volume drivers for extensibility
- Snapshot and backup
- Storage quotas
- Encryption at rest

## Consequences
- Flexible persistence
- Data portability
- Storage management complexity
