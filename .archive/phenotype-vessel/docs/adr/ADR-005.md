# ADR-005: Development Workflow Integration

## Status
**Accepted**

## Context
Vessel must integrate seamlessly with developer workflows.

## Decision
- File watch and auto-restart
- Log streaming
- Interactive shell access
- Debug port forwarding
- IDE integration
- CI/CD pipeline support

## Consequences
- Fast iteration
- Good debugging
- Some resource overhead
