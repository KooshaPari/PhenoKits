# Support

## Getting Help

Use the channel that matches the kind of help you need:

- Questions about usage, setup, or pack authoring: open a GitHub Discussion.
- Reproducible bugs or regressions: open a GitHub Issue with steps to reproduce.
- Security concerns: follow the private reporting process in [SECURITY.md](SECURITY.md).

## Before You Ask

Collect the relevant context first:

- DINOForge version from `VERSION`
- whether you are using `src/DINOForge.CI.sln` or `src/DINOForge.sln`
- the pack IDs involved, if the issue is content-related
- logs, stack traces, or failing commands
- whether the issue reproduces on the current `main` branch

## What Maintainers Need

Include enough detail to make the report actionable:

- expected behavior
- actual behavior
- exact commands run
- environment details such as .NET SDK version and OS
- a minimal repro pack or sample when possible

## Response Expectations

DINOForge is maintained on a best-effort basis. Security reports are handled under the timelines in [SECURITY.md](SECURITY.md). General support and bug triage depend on maintainer availability.
