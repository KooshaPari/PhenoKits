4. Manifests update automatically

API ENDPOINTS:
- GET /v3/models/{uid}/ → Get model metadata
- GET /v3/models/{uid}/download → Get download URLs (must use immediately)
- GET /v3/search?q=query → Search for models

IMPORTANT NOTES:
- S3 signed URLs expire in 5 minutes - download starts immediately
- Free API tier: 100-200 calls/day (each model needs 2 calls)
- Some models restricted by creator (403 Forbidden) - unavoidable
- Rate limiting: 1-2 requests/second recommended
- Script automatically retries on 429 (Too Many Requests)

TROUBLESHOOTING:
- 403 Forbidden: Creator restricted, find alternative model
- 429 Too Many Requests: Rate limited, wait or get personal token
- 404 Not Found: Invalid model ID, verify with API

For detailed API documentation, see: SKETCHFAB_DOWNLOADER_REPORT.md
