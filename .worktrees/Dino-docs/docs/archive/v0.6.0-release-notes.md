## What's New in v0.6.0

### ✅ Complete Gameplay System
- **26 units** fully defined (Republic + CIS) with complete stats, costs, and vanilla mappings
- **20 buildings** fully defined (command, barracks, defense, economy, research) with production chains
- **14 weapon types** with faction-specific variants (blasters, sniper rifles, rotary cannons, lightsabers, etc.)
- **6 doctrines** with complete mechanics (Elite Discipline, Jedi Leadership, Defensive Formation, Mechanized Attrition, Rolling Thunder, Swarm Protocol)
- **Wave system** with campaign-ready templates for Clone Wars scenarios
- **Full faction differentiation**: Republic (elite, disciplined, hero-focused) vs CIS (swarm, droid-based, mass production)

### ✅ Complete Definitions & Mechanics
All 100% field-complete as verified in DEFINITIONS_COMPLETENESS_AUDIT.md:
- Unit definitions: id, display_name, description, unit_class, faction_id, tier, weapon, vanilla_mapping, defense_tags, behavior_tags, stats
- Building definitions: id, display_name, description, building_type, health, cost, production
- Weapon definitions: 14 types, each with unique damage profiles and ranges
- Balance system: 3-tier progression (Tier 1 basic, Tier 2 intermediate, Tier 3 elite)
- Resource economy: Food, Wood, Stone, Iron, Gold, Population resources fully tracked

### ✅ VFX & Visual Systems
- Faction-aware particle effects (Republic blue #4488FF, CIS orange #FF4400)
- VFX pooling with zero-allocation design (48 instance pool)
- LOD management (FULL/MEDIUM/CULLED distance-based culling)
- Impact VFX for projectiles, death VFX for units, collapse VFX for buildings
- All systems tested and working (664 tests passing)

### ✅ Asset Pipeline
- Sketchfab API integration for automated model downloads
- FBX/GLB extraction from ZIP archives
- Asset manifest generation with attribution
- 4 models downloaded and integrated:

  - Droid Starfighter (CIS aircraft) - 1.3MB
  - Stormtrooper (placeholder, will be replaced) - 704KB
  - Clone Trooper Helmet (partial, will be completed) - 137KB

- **664 unit tests passing** (100% pass rate)
- **20 property-based tests** (randomized validation)

- Zero critical bugs

- Clone Trooper model is helmet-only, not full body

## Installation & Usage

### Requirements
- DINOForge framework v0.6.0+
- DINO game (Unity 2021.3.45f2, ECS enabled)
- BepInEx 5.4.23.5+

### Installation
```bash
# Clone or download
git clone https://github.com/KooshaPari/Dino.git
cd Dino

# Build
dotnet build src/DINOForge.sln

# Run tests (verify everything works)
dotnet test src/DINOForge.sln

# Install pack to game
copy packs/warfare-starwars/pack.yaml <DINO_GAME>/BepInEx/plugins/DINOForge/
```

### Modding
```bash
# Use PackCompiler to validate
dotnet run --project src/Tools/PackCompiler -- validate packs/

# Build custom pack
dotnet run --project src/Tools/PackCompiler -- build packs/<your-pack>
```
