## Sourcing Criteria

### Mandatory Requirements
- **License**: Must be CC0, CC-BY, CC-BY-SA, or commercial-friendly (check model page)
- **Format**: GLB/FBX export available
- **Quality**: Minimum 1000 triangles, maximum 50,000 (for LOD purposes)
- **Theme**: Must fit Star Wars Clone Wars aesthetic (Geonosis-style, Coruscant, Naboo, Tanaab)
- **Completeness**: Must be complete/usable as-is (not WIP/concept)

### Preference Ranking
1. **Fan-created Star Wars (licensed)** — High canon fidelity
2. **Original Geonosis/Clone Wars aesthetic** — Perfect thematic fit
3. **Generic sci-fi military** — Can be reskinned/themed
4. **Generic fantasy** — Last resort (poorest fit)

### Red Flags (Skip if):
- Model requires permission/PSD for commercial use
- Model has broken geometry or missing materials
- No clear license statement
- Model is explicitly "not for commercial/game use"
- Requires Patreon/subscription download
