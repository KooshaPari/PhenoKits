## DIAGNOSTIC DECISION TREE

IF SetPacks not in log:
  → SetPacks() is never being called
  → Check: ModMenuOverlayProxy and pack loading system
  → Check: When is SetPacks called vs when is DFCanvas ready?

ELSE IF _listContent: NULL in log:
  → UI initialization incomplete when SetPacks() was called
  → Check: DFCanvas.Start() completion timing
  → Check: Is Build() being called?

ELSE IF items created but not visible AND sizeDelta=(0, 40):
  → Items ARE created but have zero height
  → Problem: VerticalLayoutGroup forcing height to 0
  → Or: ContentSizeFitter not expanding container
  → Or: Items are created in wrong parent

ELSE IF items created with correct sizeDelta=(200, 17):
  → Items have correct size but invisible
  → Check: Parent GameObjects are active (gameObject.activeSelf)
  → Check: Text colors have non-zero alpha
  → Check: Canvas rendering enabled
  → Check: Sorting order correct (should be 100)
  → Check: Text.font is not null
