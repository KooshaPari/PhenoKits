## Download Instructions

### For Each Asset

```bash
# 1. Navigate to asset directory
cd packs/warfare-starwars/assets/raw/{asset_id}/

# 2. Read download URL from asset_manifest.json
# 3. Visit Sketchfab URL manually OR use:
curl -L "https://sketchfab.com/download/{source_model_id}" \
  -H "Accept: application/zip" \
  -o source_download.zip

# 4. Extract GLB file if necessary
# 5. Compute SHA256 hash
## Windows:
certutil -hashfile source_download.glb SHA256

## Linux/macOS:
sha256sum source_download.glb

# 6. Update asset_manifest.json:
#    - Add "sha256": "computed_hash_value"
#    - Set "download_status": "downloaded"
#    - Update "acquired_at_utc" to ISO 8601 timestamp

# 7. Validate schema
dotnet run --project src/Tools/PackCompiler -- \
  validate-asset raw/{asset_id}/asset_manifest.json
```
