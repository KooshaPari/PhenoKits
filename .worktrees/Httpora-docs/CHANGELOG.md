# Changelog

All notable changes to Httpora are documented here.

## [0.1.0] - 2026-04-02

### Added

- Initial release
- HTTP client/server toolkit with middleware support
- Tower-compatible async HTTP middleware
- Rate limiting (token bucket and fixed-window)
- Retry logic with exponential backoff and jitter
- Circuit breaker pattern implementation
- CORS helpers
- Request/response builders
