# Changelog

All notable changes to Zerokit are documented here.

## [0.1.0] - 2026-04-02

### Added

- Initial release
- Zero-config cryptography toolkit
- SHA-256/512 hashing with hex encoding helpers
- HMAC with SHA256/512
- AES-GCM authenticated encryption
- JWT signing/verification (HS256, Ed25519, RS256)
- Key derivation (Argon2id, PBKDF2-HMAC-SHA256)
- Constant-time operations to prevent timing attacks
- Zero unsafe code policy
