# AgentOps Policy Federation

Centralized governance layer for multi-agent operations in the Phenotype organization. Provides scope contracts, federation rules, and runtime extensions for agent harnesses.

## Key Concepts

- **Scope federation** — federated policy contracts across agent runtimes
- **Governance** — shared rules and compliance for agent devops workflows
- **Runtime extensions** — hooks for Codex, Cursor-agent, Claude, and Factory-Droid
