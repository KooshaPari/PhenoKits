# devenv-abstraction

> **⚠️ DEPRECATED**: This project has been renamed to **[NanoVMS](https://github.com/KooshaPari/nanovms)**

NanoVMS (Nano Virtual Machine Services) provides the same functionality with an updated architecture and improved platform support.

## Migration

Please use the new repository:

```bash
# Clone the new repository
git clone https://github.com/KooshaPari/nanovms.git
cd nanovms
```

## What's Changed

| Old (devenv-abstraction) | New (NanoVMS) |
|--------------------------|---------------|
| `devenv-abstraction` | `nanovms` |
| Platform-specific flavors | Infrastructure layer (Lima, WSL, Native, MicroVM, WASM) |
| Limited platform targets | Full platform roadmap (Apple, Android, AR/VR, Gaming, IoT) |

## Documentation

All documentation has moved to the [NanoVMS repository](https://github.com/KooshaPari/nanovms).
