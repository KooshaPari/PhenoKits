# hexagon-ts Plan

## Overview
Hexagonal Architecture template for TypeScript/Node.js.

## Phases

### Phase 1: Core Structure (1 week)
- Domain layer (entities, services)
- Application layer (use cases, interfaces)
- Adapter layer structure

### Phase 2: Adapters (2 weeks)
- Primary adapters (Express, CLI)
- Secondary adapters (database, external APIs)
- Interface implementations

### Phase 3: Integration (1 week)
- Dependency injection wiring
- Working examples
- Documentation

## Deliverables
- TypeScript hexagonal template
- Domain layer with DDD patterns
- Express adapter example
- Unit tests with vitest

## Resources
- 1 TypeScript developer
- Dependencies: pnpm, vitest, express
