# Phenotype Agent Core

Multi-Agent Orchestration Platform with LLM Integration

## Features

- **Multi-Provider LLM Support**: OpenAI, Anthropic, Google, Ollama
- **Skill System**: Extensible, hot-reloadable skills
- **Tool Integration**: Automatic schema generation
- **Agent Types**: Specialized agents for different tasks
- **Task Engine**: Async orchestration with dependencies

## Quick Start

```bash
# Install
pip install phenotype-agent-core

# Run agent
agent-core run "What is the capital of France?"

# Interactive mode
agent-core shell
```
