# Security Policy

## Reporting Security Vulnerabilities

If you discover a security vulnerability, please do **NOT** open a public issue. Instead:

1. Go to the [Security Advisories](https://github.com/KooshaPari/phenotype-gauge/security/advisories) page
2. Click "Report a vulnerability"
3. Provide details about the issue

We aim to respond within 48 hours and will work with you to understand and address the issue promptly.

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.2.x   | :white_check_mark: |
| < 0.2   | :x:                |

## Security Best Practices

When using this library:

- Keep dependencies updated
- Review dependency changes before updating
- Use pinned versions in production
