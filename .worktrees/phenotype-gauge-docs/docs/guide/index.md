# Guide

`phenotype-gauge` packages benchmarking and xDD testing around a small set of
repeatable workflows.

## Core flows

- Start with [Getting Started](/getting-started) for installation and first-run examples.
- Use the reference section to understand module boundaries and public concepts.
- Keep benchmarking and test-quality concerns separate from runtime observability.
