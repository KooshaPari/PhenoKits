# Phenotype-gauge docs site completion

## Goal
Bring the phenotype-gauge docs site and companion validation surface up to a complete, automated state.

## Scope
- Confirm the docs site builds cleanly.
- Repair or add missing docs routes and entry pages.
- Add navigation for all public docs pages so key content is reachable.
- Add baseline validation coverage for docs build and route health.
- Fix any broken build, parser, or test gaps discovered during audit.

## Acceptance Criteria
- Docs site builds without fatal errors.
- Public docs routes resolve and the core doc pages are reachable.
- Automated docs checks exist for the primary public paths.
- Changes stay isolated from unrelated repository work.
