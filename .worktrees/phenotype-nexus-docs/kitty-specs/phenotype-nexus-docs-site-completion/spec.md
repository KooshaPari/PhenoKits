# phenotype-nexus docs site completion

## Goal
Bring the phenotype-nexus documentation site to a complete, validated state with stable routes and test coverage.

## Scope
- Audit the docs site for missing public pages and broken routes.
- Add locale entry points or route coverage if they are missing.
- Add boilerplate validation for docs build, unit, component, and e2e checks.
- Keep the docs surface aligned with the archived crate documentation.

## Acceptance Criteria
- Docs site builds successfully.
- Public docs routes resolve reliably.
- Automated docs checks exist for the public surface.
- Changes stay within the docs and validation layers.
