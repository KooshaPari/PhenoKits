# Phenotype Templates

> Unified template registry for the Phenotype ecosystem

## Overview

This repository consolidates all project templates into a single, searchable registry with multi-language support.

## Templates

| Template | Language | Framework | Status |
|----------|----------|-----------|--------|
| python | Python | Poetry + FastAPI | Stable |
| rust | Rust | Cargo + Axum | Stable |
| go | Go | Standard + Gin | Stable |
| typescript | TypeScript | Node + Express | Stable |
| kotlin | Kotlin | Gradle + Spring | Beta |
| swift | Swift | SPM + Vapor | Beta |
| zig | Zig | Build.zig | Alpha |
| mojo | Mojo | MAX | Alpha |
| elixir | Elixir | Mix + Phoenix | Alpha |
