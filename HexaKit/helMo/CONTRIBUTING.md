# Contributing

## Local development

Clone the repo and work on a branch as usual. Keep dependencies and tooling aligned with what this repository uses once they are defined in the root (for example lockfiles or config files).

## Remote

**GitHub:** https://github.com/KooshaPari/helMo — default branch **`main`**. After clone, `git remote -v` should show `origin` pointing at this URL.

## Scratch and temporary files

Use **`.tmp/`** at the repository root for local scratch, experiments, and throwaway artifacts. It is listed in `.gitignore` and must not be committed.

## Agents and automation

If you use AI agents or automated tooling in this tree, read **[AGENTS.md](./AGENTS.md)** for project-specific rules and conventions.
