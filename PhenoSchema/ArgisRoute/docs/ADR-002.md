# ADR-002: Middleware Architecture

## Status
Accepted

## Context
Cross-cutting concerns require:
1. Authentication/authorization
2. Logging and tracing
3. Rate limiting
4. Request transformation

## Decision
Implement chain-of-responsibility middleware:
1. Middleware as functions
2. Execution order guarantees
3. Context for data sharing
4. Early termination support

## Implementation

```go
type Middleware func(HandlerFunc) HandlerFunc

type HandlerFunc func(Context) error

func LoggingMiddleware(next HandlerFunc) HandlerFunc {
    return func(c Context) error {
        start := time.Now()
        err := next(c)
        
        slog.Info("request",
            "path", c.Request().URL.Path,
            "duration", time.Since(start),
            "error", err,
        )
        
        return err
    }
}
```
