# ADR-003: Load Balancing Integration

## Status
Accepted

## Context
Routing requires:
1. Multiple backend support
2. Health-aware routing
3. Load distribution
4. Failover handling

## Decision
Integrate load balancing with:
1. Round-robin as default strategy
2. Health check integration
3. Weighted distribution support
4. Circuit breaker pattern

## Implementation

```go
type LoadBalancer interface {
    Select(backends []Backend) Backend
}

type RoundRobin struct {
    counter uint64
}

func (r *RoundRobin) Select(backends []Backend) Backend {
    if len(backends) == 0 {
        return nil
    }
    
    n := atomic.AddUint64(&r.counter, 1)
    return backends[n%uint64(len(backends))]
}
```
