# Changelog

All notable changes to Stashly will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.0.1] - 2026-04-24

### Added

- Initial bootstrap of Stashly collection
- `stashly-cache`: Two-tier LRU + DashMap cache with TTL support
- `stashly-eventstore`: Append-only event store with SHA-256 hash chains
- `stashly-statemachine`: Generic FSM with transition guards and context
- Workspace structure with unified Cargo.toml
- AGENTS.md and CLAUDE.md project documentation
- docs/EXTRACTION_PLAN.md with provenance notes
