# Python Project Template

> Cookiecutter template for Python projects in Phenotype ecosystem

## Overview

Standardized Python project template with best practices, tooling, and CI/CD configuration.

## Features

- **Poetry**: Modern dependency management
- **Ruff**: Fast Python linter
- **pytest**: Testing framework
- **GitHub Actions**: CI/CD pipeline
- **Pre-commit Hooks**: Code quality checks

## Usage

```bash
# Install cookiecutter
pip install cookiecutter

# Generate project
cookiecutter gh:phenotype-dev/template-python
```

## Included

- `pyproject.toml` - Modern Python packaging
- `src/` layout - Best practice structure
- `tests/` - Test configuration
- `.github/workflows/` - CI/CD setup
- `README.md` - Project template
- `LICENSE` - MIT license
