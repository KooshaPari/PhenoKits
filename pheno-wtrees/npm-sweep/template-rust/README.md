# Rust Project Template

> Cookiecutter template for Rust projects in Phenotype ecosystem

## Overview

Standardized Rust project template with best practices, tooling, and CI/CD configuration.

## Features

- **Cargo**: Native Rust tooling
- **Clippy**: Linting
- **rustfmt**: Code formatting
- **GitHub Actions**: CI/CD pipeline
- **Pre-commit Hooks**: Code quality

## Usage

```bash
# Install cookiecutter
pip install cookiecutter

# Generate project
cookiecutter gh:phenotype-dev/template-rust
```

## Included

- `Cargo.toml` - Rust manifest
- `src/` - Source code
- `tests/` - Integration tests
- `.github/workflows/` - CI/CD
- `README.md` - Project template
- `LICENSE` - MIT license
