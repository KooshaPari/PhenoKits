# ForgeCode Fork

> Fork of forgecode with custom modifications

## Overview

This is a specialized fork of the forgecode project with additional features and customizations for the Phenotype ecosystem.

## Features

- **Extended Templates**: Additional code generation templates
- **Custom Transformers**: Phenotype-specific transformations
- **Integration Hooks**: Better integration with Phenotype tools

## Quick Start

```bash
# Install
pip install forgecode-fork

# Generate code
forgecode generate --template phenotype-api --output ./api
```
