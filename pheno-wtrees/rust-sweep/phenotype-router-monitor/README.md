# Phenotype Router Monitor

> Network routing monitoring and diagnostics

## Overview

Router monitoring tool for tracking network routes, performance, and health across Phenotype infrastructure.

## Features

- Route tracking
- Latency monitoring
- Health checks
- Alert integration

## Quick Start

```bash
# Monitor routes
router-monitor watch --interval 30s

# Check status
router-monitor status
```
