# ADR-001: Package Distribution System

## Status
**Accepted**

## Context
The Phenotype ecosystem needs a unified package distribution system for templates, configurations, and binary artifacts.

## Decision
- Registry-based distribution
- Semantic versioning
- Multi-format support (tar, zip, OCI)
- Checksum verification
- Signature validation (Sigstore)
- CDN distribution

## Consequences
- Secure distribution
- Fast downloads
- Complex infrastructure
