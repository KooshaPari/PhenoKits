# ADR-002: Versioning and Release Strategy

## Status
**Accepted**

## Context
Package versioning affects compatibility and update strategies.

## Decision
- SemVer 2.0 strictly enforced
- Pre-release version support
- Lock file for reproducibility
- Update channel (stable, beta, nightly)
- Breaking change detection
- Migration guide requirements

## Consequences
- Predictable updates
- Breaking change management
- Migration effort tracking
