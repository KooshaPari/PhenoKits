# ADR-003: Package Registry Architecture

## Status
**Accepted**

## Context
The registry must handle metadata, storage, and search for packages.

## Decision
- PostgreSQL for metadata
- S3-compatible storage for artifacts
- Redis for caching
- Elasticsearch for search
- GraphQL API
- Webhook notifications

## Consequences
- Scalable architecture
- Fast search
- Complex stack
