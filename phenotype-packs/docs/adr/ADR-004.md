# ADR-004: Dependency Resolution

## Status
**Accepted**

## Context
Packages depend on other packages; resolution must be deterministic.

## Decision
- Lock file format
- Minimal version selection
- Conflict resolution rules
- Circular dependency detection
- Peer dependency support
- Workspace support

## Consequences
- Reproducible installs
- Deterministic builds
- Resolution complexity
