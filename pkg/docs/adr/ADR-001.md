# ADR-001: Package Manager Architecture

## Status
**Accepted**

## Context
Unified package management across Phenotype projects.

## Decision
- CLI-first design
- Multi-language support
- Lock file for reproducibility
- Global and project-local installs
- Script running capability
- Dependency tree visualization

## Consequences
- Consistent package management
- Learning curve
- Maintenance burden
