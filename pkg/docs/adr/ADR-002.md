# ADR-002: Resolution Algorithm

## Status
**Accepted**

## Context
Deterministic dependency resolution is critical for reproducible builds.

## Decision
- SAT solver approach
- Minimal version selection (Go-style)
- Constraint propagation
- Conflict explanation
- Override support
- Lock file generation

## Consequences
- Reproducible resolution
- Complex algorithm
- Good error messages
