# ADR-003: Registry Protocol

## Status
**Accepted**

## Context
Standard protocol for package registry communication.

## Decision
- HTTP/2 for efficiency
- JSON API
- Content-addressable storage
- Checksum verification
- CDN-friendly caching
- Authentication via tokens

## Consequences
- Fast downloads
- Secure distribution
- Standard infrastructure
