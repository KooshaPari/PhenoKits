# ADR-004: Build from Source

## Status
**Accepted**

## Context
Some packages need compilation for the target platform.

## Decision
- Source package specification
- Build script standardization
- Cache compiled artifacts
- Cross-compilation support
- Build environment isolation
- Build dependency management

## Consequences
- Platform-native performance
- Build time overhead
- Complex build orchestration
