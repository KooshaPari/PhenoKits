# ADR-005: Workspace Support

## Status
**Accepted**

## Context
Monorepos need workspace-aware package management.

## Decision
- Workspace manifest
- Member package discovery
- Shared dependency resolution
- Cross-member dependency linking
- Unified version bumping
- Workspace-wide commands

## Consequences
- Monorepo-friendly
- Complex workspace logic
- Good for large projects

1. [Cargo Workspaces](https://doc.rust-lang.org/cargo/reference/workspaces.html)
