# ADR-003: Security Model

## Status
**Accepted**

## Context
Plugins run untrusted code; security is paramount.

## Decision
- WASM sandboxing
- Capability-based access control
- Resource limits (CPU, memory, time)
- No filesystem access without permission
- Network access control
- Code signing requirement

## Consequences
- Defense in depth
- Plugin capability limitations
- Trust establishment needed
