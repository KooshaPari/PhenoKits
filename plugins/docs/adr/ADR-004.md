# ADR-004: Plugin Lifecycle

## Status
**Accepted**

## Context
Plugins need well-defined lifecycle for predictable behavior.

## Decision
- Discovery phase
- Initialization with configuration
- Runtime execution
- Health checking
- Graceful shutdown
- Cleanup and resource release

## Consequences
- Predictable plugin behavior
- Resource cleanup
- Complexity in state management
