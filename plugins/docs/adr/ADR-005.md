# ADR-005: Plugin Distribution

## Status
**Accepted**

## Context
Plugins need secure distribution and update mechanism.

## Decision
- Central plugin registry
- Version management
- Automatic updates (configurable)
- Rollback capability
- Plugin verification
- Dependency resolution for plugins

## Consequences
- Easy plugin discovery
- Update management
- Registry infrastructure needed
