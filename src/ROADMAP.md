# src/ Roadmap

**Roadmap**

This document tracks large features on the engineering roadmap. This is not an authoritative document - we have internal project management. It's mainly here to reduce questions.

It should not be assumed that the architecture is 'set' based on this roadmap. We have many other features that might appear down the road.

## Current (Q2 2026)

- Stdio transport GA
- HTTP transport (in progress)
- Tool registry with hot-reload
- Protocol compliance (beta)

## Near-term (Q3 2026)

- WebSocket transport
- Resource system
- Multi-tool batch operations
- HTTP transport GA

## Future (Q4 2026+)

- Prompt templates
- Sampling API
- Distributed caching
- Advanced observability
- Tool marketplace integration
